jackfish

A javascript chess engine inspired by sunfish.

jackfish
Author: Mattias Ahlsén (mattias.ahlsen@gmail.com)
Example
import Engine from 'jackfish';
const game = new Engine();

Options

Configuration object for the engine.

Options
Properties
startPos (string?) : The FEN string of the starting position.
Example
const options = {
  startPos: 'rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1',
}

Engine

The chess engine, including game mechanics and a configurable AI. (default export)
